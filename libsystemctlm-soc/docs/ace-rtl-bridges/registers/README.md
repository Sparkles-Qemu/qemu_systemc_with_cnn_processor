## Acronyms used for Register types
   
NOTE : Register Addresses are in Little Endian Byte Order.   
    
wo 	: write only.  
ro 	: read only.  
rw 	: read and write.  
w1sraz 	: write 1 to set matching bit, write 0 has not effect and read will always give 0.
